Getting Started

Manage your web service requests with the four basic methods of the standard Restful get, post, put and delete (currently, only implemented GET method).

With the standard anotation system HATEOAS incorporated in each of its resources, you can access not only the resource itself but any of the API that is related to it.

This tool is developed as an academic and learning use. The objective is not a commercial use.

Please be patient with its execution and the possible errors and incidents that may appear.

Thanks for your understanding and patience.

Getting Category by ID

To get the information of a specific category, just add your id number of category identification after /api/categorias/. This option does not support filters or search conditions.
Access with the get () method.

http://server/api/categorias/3.

Get articles list

Get the list of existing movies in the database. We can include different selection criteria indicating the 'start date' of the search, the 'end date' of the search or a certain 'number' of last articles add in the database.All options are accessible through the link /api/articulos/ in the url.
Access with the get () method.

To make filters it will be necessary to indicate:

/api/articulos/?min="dd-mm-yyyy" List movies added from the date indicated

http://server/api/articulos/?min="10-05-2018"

/api/articulos/?max="dd-mm-yyyy" List movies added after the date indicated

http://server/api/articulos/?max="19-05-2018"

/api/articulos/?min="dd-mm-yyyy"&max="dd-mm-yyyy" List movies added between the dates indicated

http://server/api/articulos/?min="10-05-2018"&max="19-05-2018"

/api/articulos/?n="n" List the 'n' latest movies add in the database

http://server/api/articulos/?n="5"

http://server/api/articulos/
